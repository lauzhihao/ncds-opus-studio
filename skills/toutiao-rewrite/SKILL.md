# 头条图文改写

## 使用方式

提供原始素材（文本、转写稿、新闻摘要等），按头条平台规范改写为图文文章。

### 示例

```
把这段内容改写成头条文章：（粘贴内容）
```

```
根据这个视频转写稿，写一篇头条图文
```

## 触发关键词

- "头条文章"
- "头条图文"
- "头条改写"
- "toutiao rewrite"

## 规范

参考 `references/toutiao-article-rules.md` 中的完整写作规范。

## 输出格式

```
【标题】吸引点击但不低俗的标题
【正文】
  - 开头：抛出核心信息或悬念
  - 正文：分段叙述，每段 2-3 句
  - 结尾：总结观点或引导互动
【配图建议】描述需要什么样的配图
```
