# 抖音文案改写

## 使用方式

提供原始文案或转写文本，按抖音平台规范改写为短视频文案。

### 示例

```
把这段转写文本改写成抖音文案：（粘贴文本）
```

```
帮我写一条关于 xxx 的抖音文案
```

## 触发关键词

- "抖音文案"
- "改写文案"
- "短视频文案"
- "douyin copy"

## 规范

参考 `references/douyin-copy-rules.md` 中的完整写作规范。

## 输出格式

```
【钩子/开头】一句话抓住注意力
【正文】口语化短句，节奏感强
【互动引导】引导评论/点赞/关注
【话题标签】#相关话题 #热门标签
```
